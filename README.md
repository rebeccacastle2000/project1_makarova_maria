# Лабиринт сокровищ

Текстовая приключенческая игра на Python. Исследуйте древний лабиринт, решайте загадки, избегайте ловушек и найдите сокровище!

## Особенности игры
- 6 уникальных комнат с описаниями, предметами и загадками:
    - entrance — темный вход с факелом
    - hall — зал с загадкой числа
    - trap_room — комната с ловушкой и плитами
    - library — библиотека со свитками и загадкой
    - armory — оружейная с мечом и шкатулкой
    - treasure_room — комната сокровищ (финал)
- Динамические события:
    - Случайные находки (монетки)
    - Шорох в темноте (отпугивается мечом)
    - Ловушки, активируемые при отсутствии факела
- Система инвентаря:
    - Поднятие предметов (torch, sword, ancient_book, bronze_box)
    - Использование предметов с уникальными эффектами
    - Ржавый ключ (rusty_key) для доступа в сокровищницу
- Загадки с альтернативными ответами:
    - 10 или десять
    - шаг шаг шаг или шагшагшаг
    - Два пути к победе:
    - Найти treasure_key (решив загадку в hall) и открыть сундук
    - Взломать сундук кодом (решив загадку комнаты)

## Условия проигрыша
    1. Игра завершается поражением в следующих случаях:
    2. Ловушка в темноте: Неправильно решая загадку в trap_room без факела (torch), вы активируете ловушку — мгновенное поражение.
    3. Исчерпание попыток: 3 неверных ответа на одну загадку — автоматическое поражение.
    4. Ловушка с пустым инвентарём: При срабатывании ловушки без предметов есть 30% шанс гибели.


## Требования

- Python 3.13+
- Poetry 1.8.4+

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/ваш-логин/project1-makarova-maria.git
cd project1-makarova-maria

# Установка зависимостей
make install
# или
poetry install
```

## Запуск
```bash
# Запуск игры
make project
# или
poetry run project
```

## Управление
```bash
go <direction>      - перейти в направлении (north/south/east/west)
look                - осмотреть текущую комнату
take <item>         - поднять предмет
use <item>          - использовать предмет из инвентаря
inventory           - показать инвентарь
solve               - попытаться решить загадку в комнате
help                - показать список команд
quit                - выйти из игры

```
